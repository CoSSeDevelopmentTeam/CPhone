Nukkit向け疑似スマートフォンシステム

# 特徴
✔ ライフサイクルを用いたフォームの設計  
✔ フォームの多言語対応  
✔ 各機能は独立したjarファイルにて定義  
✔ 自動化されたフォームの管理  
✔ 非常に簡単なしふぉんアプリケーションの開発  
  
# 操作
ガストの涙（デフォルト）を手にもって空中をタップするとしふぉんが開きます。  
  
詳細は[Wiki](https://github.com/CoSSeDevelopmentTeam/CPhone/wiki)をご覧ください。
